Laboratory Work

Lab 1: User Interface (SDG 16)

Building the foundational transparent UI for a civic grievance reporting tool.

Reflection: Building the initial UI using Jetpack Compose helped me understand how to design accessible public platforms. I focused on clear, unbiased layouts crucial for inclusive institutional applications.

Lab 2: User Interaction (SDG 16)

Implementing reliable state management for citizens to securely log community issues.

Reflection: Implementing state hoisting was vital for data integrity. I learned how to manage user inputs dynamically, ensuring citizens can reliably submit reports to authorities without frustrating data loss.

Lab 3: Material Design (SDG 16)

Refining the visual architecture to project authority, trust, and institutional reliability.

Reflection: Integrating specific SDG 16 themes required a clean, authoritative UI design. I utilized Compose's theming capabilities to enforce consistent typography, colors, and accessibility standards to foster public trust.

Lab 4: Navigation (SDG 16)

Constructing a robust navigation graph connecting citizens to various justice and reporting modules.

Reflection: Setting up the Navigation component simplified moving between complex screens. I ensured a seamless user experience so that navigating legal and civic reporting procedures feels intuitive and transparent.

Lab 5: Local Database (SDG 16)

Integrating secure local storage to ensure equitable offline access for incident reporting.

Reflection: Implementing Room allowed the app to cache user reports offline. This is critical for justice equity, allowing citizens in low-connectivity rural areas to draft complaints and sync them when they reach a network.
